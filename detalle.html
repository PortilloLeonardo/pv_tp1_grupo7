Proyecto: Conectividad Educativa Rural

Fecha de inicio: 15 de Marzo de 2026 | Autores: Grupo 7

Descripción Técnica del Proyecto

1. Resumen Ejecutivo Este proyecto aborda la falta de acceso digital en escuelas rurales del norte argentino mediante el despliegue de infraestructuras híbridas que combinan enlaces de larga distancia con redes internas de alta eficiencia. El objetivo es proporcionar un entorno de red seguro y optimizado para el consumo de contenidos pedagógicos de alta demanda.

2. Arquitectura de Red y Topología La infraestructura se divide en tres capas fundamentales, diseñadas para maximizar la estabilidad en terrenos de difícil acceso: *Capa de Acceso (WAN): Implementación de enlaces satelitales y radioenlaces de microondas para superar las barreras geográficas del relieve montañoso. *Capa de Distribución: Segmentación de red mediante VLANs para separar el tráfico administrativo del tráfico de los estudiantes, garantizando la integridad de los datos institucionales. *Capa Local (LAN): Configuración de switches con protocolos de enrutamiento estático para minimizar la carga de procesamiento en los nodos locales y asegurar una latencia baja.

3. Especificaciones de Configuración y Seguridad *El núcleo del sistema utiliza herramientas de código abierto para permitir la escalabilidad del proyecto: *Gestión de Tráfico: Uso de firewalls para el filtrado de contenido y prevención de intrusiones, asegurando un entorno protegido para menores. *Priorización (QoS): Configuración de Calidad de Servicio para dar prioridad absoluta a plataformas de videoconferencia y repositorios educativos sobre otros tipos de tráfico menos críticos. *Optimización de Ancho de Banda: Implementación de servidores caché locales para almacenar contenido educativo frecuente, reduciendo la dependencia constante del enlace satelital.

4. Componentes de Hardware Utilizados Switches: Equipos configurables para el soporte de las topologías diseñadas en los manuales técnicos. Access Points: Puntos de acceso de alta ganancia para cubrir áreas comunes y aulas rurales. Nodos de Energía: Sistemas de alimentación ininterrumpida (UPS) o paneles solares (según disponibilidad) para mantener la red operativa ante cortes eléctricos.

5. Documentación de Soporte Para la replicabilidad del modelo, se han desarrollado los siguientes documentos técnicos: Manual de Configuración de Switches: Guía paso a paso para la implementación de la lógica de red y esquemas de topología. Esquema de Topología: Diagramas visuales que detallan la interconexión de cada nodo en el entorno rural. 3. Especificaciones de Configuración y Seguridad
